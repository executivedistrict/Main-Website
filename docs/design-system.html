Design System

Executive District visual reference

Light system recreated 1:1 from the production site (snapshots in docs/Original Site HTML/). White pages, navy anchor surfaces, blue for every primary action, gold strictly as an accent. Inter for all text; Lora italic appears only inside headlines via <em> and in a few serif flourishes (step numbers, pull quotes). Live tokens: src/app/globals.css. Primitives: src/components/ui/.

Typography

Inter carries everything: headlines, body, UI, the uppercase wordmark. Lora italic exists only for accents. An <em> inside an h1/h2/h3 renders Lora italic gold automatically (base CSS); reproduce the production placement exactly.

Display XL with accent

.display-xl - clamp(36px, 4.5vw, 52px) / 1.1 / 700 - page heroes

Display L with accent

.display-l - clamp(30px, 3.5vw, 40px) / 1.15 / 700 - major section heads

Display M

.display-m - clamp(24px, 2.6vw, 32px) / 1.2 / 700 - section headers

Eyebrow label

.eyebrow-label - 11px / 600 / 4px tracking / uppercase / gold - on any surface

Lead paragraph: 17px slate, line-height 1.7. Used for hero subs and section descriptions.

Body: 15-16px charcoal, line-height 1.6-1.8. Long-form legal prose uses 15px / 1.8.

Buttons

Blue is the primary action everywhere on light surfaces. The gold button appears only on navy CTA bands. Radius 4px, weight 600, subtle lift on hover.

Cards, steps, stats

Cards are white with a line border, radius 8px, lift + shadow on hover; featured adds a 3px gold top rule. Process steps sit on mist with a gold top border and an oversized Lora numeral in ice. Stats are gold figures over navy labels (frost on navy), always paired with a claims disclaimer.
