# Deferred plans

Plans drafted but not yet scheduled to execute. They've been scoped (Phase 0
contracts, recommendations.md, phase files) but no agent work has started.

**Why plans land here:**

- Drafted proactively as part of analyzing a larger initiative
- Value is real but dependencies aren't yet in place
- Scope is correct but timing is wrong (e.g. waiting on a product decision)
- Risk is acceptable but other plans compete for attention

**Before starting a new plan, compare against this folder.** See
`../AGENTS.md` for the deferred → suggested promotion workflow.
